# Compliance Framework Reference

## SOC 2 (Trust Services Criteria)

### Security (Common Criteria)
- CC1.x: Control Environment
- CC2.x: Communication and Information
- CC3.x: Risk Assessment
- CC4.x: Monitoring Activities
- CC5.x: Control Activities
- CC6.x: Logical and Physical Access
- CC7.x: System Operations
- CC8.x: Change Management
- CC9.x: Risk Mitigation

### Availability
- A1.1: Capacity planning and monitoring
- A1.2: Incident handling and recovery
- A1.3: Environmental protections

### Confidentiality
- C1.1: Data identification and classification
- C1.2: Data access restrictions

### Processing Integrity
- PI1.1: Complete, accurate, and timely processing
- PI1.2: Quality assurance at processing stages

### Privacy
- P1.1: Personal information notice and consent
- P2.1: Data collection limited to purpose
- P3.1: Data retention and disposal
- P4.1: Access to personal information
- P5.1: Disclosure to third parties

## ISO 27001:2022

### Annex A Controls (93 controls, 4 themes)
- A.5: Organizational (37 controls)
- A.6: People (8 controls)
- A.7: Physical (14 controls)
- A.8: Technological (34 controls)

### Key Controls for SaaS
- A.8.8: Management of technical vulnerabilities
- A.8.9: Configuration management
- A.8.16: Monitoring activities
- A.8.24: Use of cryptography

## GDPR Requirements

- Data Protection by Design and Default (Art. 25)
- Data Processing Records (Art. 30)
- DPO Appointment (Art. 37)
- Breach Notification (Art. 33, 34)
- Data Subject Access Requests (Art. 15)
- Right to Erasure (Art. 17)
- Data Portability (Art. 20)
- Privacy Impact Assessment (Art. 35)

## HIPAA Security Rule

- Administrative Safeguards: Security management, workforce training, contingency planning
- Physical Safeguards: Facility access, workstation security
- Technical Safeguards: Access control, audit controls, integrity controls, transmission security
- Organizational: BA agreements, group health plan requirements
- Policies and Procedures: Documentation, retention (6 years)

## PCI DSS v4.0

- Build and Maintain Secure Network (Req 1-2)
- Protect Cardholder Data (Req 3-4)
- Maintain Vulnerability Program (Req 5-6)
- Implement Access Control (Req 7-8)
- Monitor Networks (Req 9-10)
- Maintain Info Security Policy (Req 11-12)
- SAQ types: A, A-EP, B, B-IP, C, C-VT, D
- Evidence must be retained for 12 months minimum
