# Product Positioning Guide

## Positioning Framework

### Key Questions
```
1. Who is the target customer? (specific, not generic)
2. What is the category? (existing or new)
3. What is the unique benefit? (what competitors can't claim)
4. Why should they believe? (proof points)
5. What's the alternative? (what they use today)
```

### Positioning Statement Template
```
For [target customer] who [need],
[product name] is a [category] that [key benefit].
Unlike [competitor/alternative], we [unique differentiator].
```

### Example
```
For mid-market SaaS companies who need to reduce customer churn,
ChurnGuard is an AI-powered retention platform that predicts
at-risk customers 30 days in advance with 95% accuracy.
Unlike generic analytics tools, we provide specific intervention
recommendations tied to measurable outcomes.
```

## Differentiation Types

| Type | Example | Strength | Sustainability |
|------|---------|----------|----------------|
| Feature | "Only product with X" | Strong short-term | Easy to copy |
| Performance | "10x faster than Y" | Medium | Hard to match |
| Ecosystem | "Integrates with Z" | Strong | Moats grow |
| Price | "50% cheaper" | Weak | Race to bottom |
| Brand | "Trusted by industry leaders" | Very strong | Takes years to build |

## Value Proposition Canvas

### Customer Profile
```
Jobs to be Done:
  Functional: What tasks do they need to accomplish?
  Social: How do they want to be perceived?
  Emotional: How do they want to feel?

Pains:
  - What frustrates them about current solution?
  - What costs (time, money, effort) do they incur?
  - What risks do they worry about?

Gains:
  - What outcomes would delight them?
  - What would make their life easier?
  - What would exceed their expectations?
```

### Value Map
```
Products & Services: What we offer
Pain Relievers: How we address each pain point
Gain Creators: How we deliver desired outcomes
```

## Positioning by Stage

| Company Stage | Positioning Priority | Example |
|---------------|---------------------|---------|
| Pre-product/market fit | Problem clarity | "Stop losing customers to churn" |
| Early growth | Category creation | "The first AI churn prediction platform" |
| Scale-up | Category leadership | "Leading churn prediction for SaaS" |
| Mature | Brand + ecosystem | "Trusted by 10,000+ companies" |

## Competitive Positioning

### Head-to-Head
When competing directly: emphasize specific advantage
```
Us vs Competitor:
- We have [feature] they lack
- We are [X] faster/cheaper/easier
- We serve [specific segment] better
```

### Category Creation
When creating new category: define the frame
```
Old way: Manual churn analysis (reactive, slow)
New way: AI-powered prediction (proactive, accurate)
```

## Positioning Testing

### Validation Questions
```
Does the positioning:
1. Differentiate from alternatives?
2. Resonate with target customers?
3. Fit within existing market category?
4. Allow for premium pricing?
5. Scale across customer segments?
```

### Testing Methods
```
Customer interviews: Read positioning statement, ask for reaction
Landing page A/B test: Compare conversion rates with different positioning
Ad testing: CTR comparison across messaging variants
Sales feedback: Does positioning help close deals?
```

## Messaging Hierarchy

```
Elevator Pitch (15 words): "We predict customer churn before it happens."
Value Prop (1-2 sentences): "AI platform that identifies at-risk customers and recommends retention actions."
Supporting Points (3-5 bullets):
- 95% prediction accuracy
- 30-day early warning
- Specific intervention recommendations
- No engineering required
Proof: Case studies, metrics, testimonials, certifications
```
