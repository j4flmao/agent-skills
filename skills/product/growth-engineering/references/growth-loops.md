# Growth Loops

## Loop Anatomy

### Components
```
Input: Action or investment that feeds the loop
Process: How input is converted to output
Output: Growth result (new users, revenue, engagement)
Feedback: Output becomes input for the next cycle
```

### Loop Metrics
```
Cycle time: Time from input to output
Amplification factor: Output/Input ratio
Conversion rate: % of each stage that proceeds
Decay rate: How fast loop loses effectiveness
```

## Growth Loop Types

### Viral Loop
```
User invites → Friend signs up → Friend invites more
K-factor = Invite rate × Conversion rate
Target: K > 1.0 for compounding growth
```

#### Design Patterns
```
Two-sided incentives: Inviter and invitee both benefit
Share of value: Sharing is part of using the product
Content sharing: User creates content that attracts new users
Network effects: More users = more value for existing users
```

#### Examples
```
Dropbox: Referral gives extra storage to both sides
Robinhood: Free stock for referral
Notion: Templates shared publicly drive signups
Calendly: Scheduling link shared in natural workflow
```

### Content Loop
```
User creates content → Content attracts visitors → Visitors become users → New users create content
Key metric: Content virality (shares per content item)
Cycle time: Content creation to organic traffic
```

#### Design Patterns
```
UGC showcase: Feature best user content on landing page
Template marketplace: Users share templates with community
SEO optimization: User-generated content ranks in search
Social sharing: Easy to share content on social platforms
```

### Sales Loop
```
Marketing attracts leads → Sales qualifies → Deal closes → Revenue funds more marketing
Key metric: CAC payback period
Cycle time: Lead to close
```

### Paid Loop
```
Revenue → Buy ads → Acquire users → Convert to paid → More revenue
Key metric: ROAS (Return on Ad Spend)
Cycle time: Ad impression to paid conversion
```

## Loop Selection

### By Product Type
```
Consumer social: Viral loop (inherent sharing)
SaaS product: Content loop (SEO) + Viral loop (collaboration)
Marketplace: Network effect loop (buyers attract sellers)
Enterprise: Sales loop (outbound required)
```

### By Growth Stage
```
Early stage: Content loop (organic, low cost)
Growth stage: Viral loop (amplify existing users)
Scale stage: Paid loop (scale proven channels)
Mature: All loops optimized
```

## Optimization Framework

### Identify Bottlenecks
```
Map the loop end-to-end
Measure conversion at each step
Find the step with highest drop-off
That step is your bottleneck
```

### Improve Conversion
```
Reduce friction at each step
  — Fewer clicks, less information required
  — Faster load times, mobile-optimized
Increase motivation
  — Better incentives, clearer value prop
  — Social proof, urgency
Improve triggers
  — Timing of invite prompts
  — Contextual in-app requests
```

### Increase Frequency
```
Shorten cycle time
  — Faster invite delivery
  — Immediate value for referrals
Increase loops per user
  — Multiple invite moments
  -– Recurring share opportunities
```
