# Metric Frameworks

## North Star Metric

### Definition
The single metric that best captures the core value customers get from the product. It correlates with long-term retention and drives organic growth.

### Criteria
```
Leading (not lagging) indicator of retention
Correlates with revenue
Understandable by the whole company
Actionable — teams can influence it
Measurable with available data
```

### Examples
```
Spotify: Time spent listening (daily active users × listening time)
Slack: Messages sent (team communication act value)
Airbnb: Nights booked (core transaction)
Dropbox: Files stored per user (storage value)
```

### Setting Your North Star
1. Identify the core action users take that delivers value
2. Validate correlation with 90-day retention
3. Define metric formula and data sources
4. Set baseline and quarterly targets
5. Communicate across the organization

## AARRR Framework

### Acquisition
```
Metrics: Signups, traffic, CAC, channel attribution, conversion rate
Goal: Drive qualified users to first product experience
Leading indicators: Signup-to-activation conversion
```

### Activation
```
Metrics: Activation rate, time-to-activation, completion rate
Goal: User experiences core value for first time
Definition: Specific action within defined time window
```

### Retention
```
Metrics: Day 1/7/30 retention, DAU/MAU ratio, churn rate
Goal: Users return and deepen product usage
Calculation: Cohort analysis with defined action periods
```

### Revenue
```
Metrics: ARPU, ARR, LTV, conversion rate, expansion MRR, churn
Goal: Convert engaged users to paying customers
Leading indicators: Feature adoption before conversion
```

### Referral
```
Metrics: Invite rate, conversion rate, K-factor, viral cycle time
Goal: Users bring new users through product experience
Required: Low-friction sharing, clear incentive
```

## Retention Cohorts

### Daily Retention
```
Day 0 (signup day): Track activation actions
Day 1: Did user return next day?
Day 3: Did user return within 3 days?
Day 7: Did user return within 7 days?
Day 30: Did user return within 30 days?
```

### Weekly Retention
```
Week 1: Did user use product in first week?
Week 2: Did user return in week 2?
Week 4: One-month retention
Week 12: Three-month retention milestone
```

### Monthly Retention
```
Month 1: First month active
Month 3: Quarter retention
Month 6: Half-year retention
Month 12: Annual retention milestone
```

### Cohort Comparison
```
Dimension: Acquisition channel (organic vs paid vs referral)
Dimension: Plan tier (free vs pro vs enterprise)
Dimension: User segment (power user vs casual)
Dimension: Activation time (activated day 0 vs day 7)
```

## Product Health Score

### Framework
```
Adoption Score: % of team using core features weekly
Engagement Score: sessions per active user per week
Retention Score: % of users active in week 4
Feature Score: % of key features adopted
NPS Score: User satisfaction rating
```

### Composite Calculation
```
Health Score = 0.3 × Retention + 0.3 × Engagement
             + 0.2 × Adoption + 0.1 × Feature + 0.1 × NPS
```
