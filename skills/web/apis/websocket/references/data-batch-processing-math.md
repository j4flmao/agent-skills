# MapReduce Math
## Mathematical Formulas
Map Function:
$$ f: (K_1, V_1) \rightarrow list(K_2, V_2) $$

Reduce Function:
$$ g: (K_2, list(V_2)) \rightarrow list(K_3, V_3) $$

Complexity Analysis:
Time Complexity = O(M * map_cost + R * reduce_cost)
