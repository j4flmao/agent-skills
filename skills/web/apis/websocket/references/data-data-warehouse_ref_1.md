## Snowflake Micro-Partitioning Architecture
```text
+----------------------------------------------------+
|                Snowflake Architecture              |
|                                                    |
|  +----------------------------------------------+  |
|  |             Cloud Services Layer             |  |
|  |  (Optimization, Metadata, Security, Access)  |  |
|  +----------------------------------------------+  |
|                                                    |
|  +----------------------------------------------+  |
|  |             Compute Layer (Virtual WH)       |  |
|  |  [ Node ] [ Node ] [ Node ] [ Node ]         |  |
|  +----------------------------------------------+  |
|                                                    |
|  +----------------------------------------------+  |
|  |             Storage Layer (Micro-partitions) |  |
|  |  [ MP 1 ] [ MP 2 ] [ MP 3 ] [ MP 4 ]         |  |
|  |  (Columnar, Compressed, Immutable)           |  |
|  +----------------------------------------------+  |
+----------------------------------------------------+
```
