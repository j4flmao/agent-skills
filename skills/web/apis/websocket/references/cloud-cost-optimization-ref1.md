# Cost Optimization Reference 1

## Spot Instance Pricing Math

Expected Cost = P_spot * T * (1 - P_interrupt) + P_ondemand * T_recovery * P_interrupt
Where:
- P_spot: Spot instance hourly price
- P_ondemand: On-demand hourly price
- T: Time running
- P_interrupt: Probability of interruption
- T_recovery: Time to recover from interruption
