# Azure Advanced Topics

## Introduction
Advanced Azure covers enterprise-scale architecture with Azure Landing Zones, Azure DevOps integration, hybrid cloud with Arc, advanced networking, and governance at scale.

## Azure Landing Zones
Enterprise-scale reference architecture for Azure adoption. Management Group hierarchy aligned to business functions. Subscription vending for automated workload deployment. Azure Policy at scale with policy initiatives. Network topology with hub-spoke architecture. Centralized logging with Log Analytics workspaces. RBAC and PIM for privileged identity management.

## Advanced Networking
Azure Virtual WAN: managed hub-spoke WAN connectivity. Azure Firewall Premium: TLS inspection, IDPS. Azure DDoS Protection Standard: application-layer DDoS defense. Private Link: private access to PaaS services. ExpressRoute FastPath: high-performance bypass for ExpressRoute. Azure Route Server: dynamic routing for network appliances.

## Hybrid Cloud with Azure Arc
Arc-enabled servers: manage on-premises and multi-cloud VMs via Azure. Arc-enabled Kubernetes: manage K8s clusters outside AKS. Arc-enabled SQL Server: manage SQL instances across environments. Azure Automanage: automated VM configuration and management. Azure Stack HCI: hyperconverged infrastructure for edge.

## Azure DevOps Integration
Azure Boards for work tracking. Azure Repos for source control. Azure Pipelines for CI/CD with multi-stage YAML. Azure Test Plans for manual and exploratory testing. Azure Artifacts for package management.

## Security at Scale
Microsoft Defender for Cloud: unified security management. Azure Sentinel: SIEM and SOAR. Conditional Access: risk-based access control. Privileged Identity Management (PIM): JIT privileged access. Azure Key Vault Managed HSM: FIPS 140-2 Level 3. Azure Policy: compliance enforcement.

## Cost Management
Azure Cost Management + Billing with budgets, alerts, and recommendations. Azure Reservations for 1-3 year commitments. Azure Savings Plan for flexible compute discount. Azure Hybrid Benefit for Windows Server and SQL Server. Azure Dev/Test pricing for non-production workloads.
