# Roadmap Strategies

## Roadmap Types

| Type | Horizon | Granularity | Audience |
|------|---------|-------------|----------|
| Strategic | 12-24 months | Themes and objectives | Leadership, Board |
| Tactical | 3-6 months | Features and epics | Engineering, Product |
| Operational | 1-3 months | Stories and tasks | Engineering team |
| Now-Next-Later | 3-9 months | Prioritized list | All stakeholders |

## Frameworks

### RICE Prioritization
```
RICE Score = (Reach × Impact × Confidence) / Effort

Reach: How many users in a time period?
Impact: 0.25x (minimal) to 3x (massive)
Confidence: 20% (low) to 100% (high)
Effort: Person-months
```

### Opportunity Scoring
```
Score = Importance - Satisfaction
Higher scores = bigger opportunities
```

### Theme-Based Roadmapping
```
Q1: Foundation (platform, auth, data)
Q2: Growth (viral features, referrals)
Q3: Monetization (billing, premium tiers)
Q4: Scale (performance, reliability)
```

## Communication

### Cadence
- Executive update: Monthly
- Stakeholder sync: Bi-weekly
- Team update: Weekly
- Public roadmap: Quarterly

### Dealing with Changes
- Communicate reprioritization decisions transparently
- Explain the "why" behind changes
- Manage expectations with reasonable timelines
- Celebrate completed milestones
- Use OKRs to connect roadmap items to business goals
