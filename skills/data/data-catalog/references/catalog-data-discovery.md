# Catalog Data Discovery

## Data Discovery Patterns
Effective data discovery enables users to find relevant datasets quickly and understand their context.

## Search Optimization
- Full-text search on dataset names, descriptions, columns
- Faceted search by source, owner, certification status
- Semantic search using embeddings and vector search
- Autocomplete and suggestion for faster discovery
- Personalized results based on user role and usage

## Popularity Scoring
- Track dataset access frequency and recency
- Weight by consumer diversity and criticality
- Combine with certification status for ranking
- Decay scores over time for stale datasets
- Account for seasonal usage patterns

## Key Points
- Implement multi-faceted search for dataset discovery
- Use popularity scoring for relevance-based ranking
- Enable collaboration through ratings and comments
- Integrate with data quality and lineage for context
- Personalize discovery based on user behavior
