# Graph Algorithms
Graph algorithms enable advanced analytics on connected data, revealing patterns, influences, and communities.

## Algorithm Categories
- Pathfinding: Shortest path (Dijkstra, A*), all paths, minimum spanning tree
- Centrality: Degree, betweenness, closeness, PageRank, eigenvector
- Community Detection: Louvain, Label Propagation, Connected Components
- Graph Embeddings: Node2Vec, GraphSAGE, knowledge graph embeddings

## Implementation
- Neo4j Graph Data Science library for in-database algorithms
- NetworkX for Python-based analysis on smaller graphs
- Spark GraphFrames for distributed graph processing

## Key Points
- Choose algorithms based on analysis goals (find paths, measure influence, detect communities)
- Match implementation to graph size and performance requirements
- Combine multiple algorithms for richer insights
- Validate algorithm results with domain expertise
- Document algorithm selection rationale
