# Logo Systems

## Logo Variations
| Variation | When to Use |
|-----------|-------------|
| Horizontal (full) | Default: websites, headers, email signatures |
| Stacked | Square spaces, app icons, social media |
| Icon only | Favicon, mobile app, small spaces |
| Monochrome | One-color printing, watermarks, embroidery |

## Clear Space Rule
- Minimum clear space = height of the logo mark
- No text, graphics, or other elements inside clear space
- More space = more perceived value

## Minimum Size
| Application | Min Width |
|-------------|-----------|
| Digital | 32px for icon, 120px for full logo |
| Print | 0.5in for icon, 1.5in for full |
| Embroidery | 1in for icon only |
