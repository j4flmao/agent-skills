# Synthesis

## Affinity Mapping Process

1. **Transcribe**: full transcript from each session
2. **Extract**: one atomic observation per sticky note (physical or Miro/Mural)
3. **Cluster**: move notes into groups — no labels yet, let patterns emerge
4. **Label**: name each cluster (e.g., "Trust concerns", "Workflow friction")
5. **Prioritize**: vote on clusters by frequency, severity, or business impact
6. **Insight**: write one sentence per cluster summarizing the pattern

Tools: Miro, Mural, FigJam, physical sticky notes + wall.

## Findings Report Structure

1. **Executive Summary**: one paragraph with the key insight and recommendation
2. **Method Overview**: participant count, segments, method, dates
3. **Participant Profile**: table of demographics and segments
4. **Key Findings**: 3–5 findings, each with:
   - Headline (one sentence)
   - Evidence (participant quotes, task metrics, screenshots)
   - Severity rating (critical, major, minor)
5. **Opportunities**: actionable recommendations mapped to findings
6. **Appendix**: full protocol, raw data, transcript excerpts
