# Rendering Performance

## Algorithms and Formulations
O(1) Cache retrieval, O(N) memory traversal.

## Data Schemas
```json
{
  "cacheName": "v1",
  "urls": ["/app.js", "/style.css"]
}
```

## Code Examples
```javascript
self.addEventListener('fetch', event => {
  event.respondWith(
    caches.match(event.request).then(response => {
      return response || fetch(event.request);
    })
  );
});
```

## Configuration Templates
```yaml
workbox:
  swDest: dist/sw.js
  clientsClaim: true
  skipWaiting: true
```

## Decision Matrices
Cache First vs Network First

## Mathematical Formulas
$$ Cache Hit Rate = \frac{Hits}{Hits + Misses} \times 100 $$

## Best Practices and Anti-patterns
Do not cache opaque responses without size limits.
