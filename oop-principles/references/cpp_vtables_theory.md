# Deep C++ OOP VTables Theory

## Virtual Method Tables (vtable) and Virtual Table Pointers (vptr)
When a class contains a virtual function, the compiler creates a vtable for that class.
The vtable is a static array of function pointers. Each element in the vtable points to the most derived implementation of a virtual function for that class.
Objects of that class contain a hidden pointer, usually called the vptr, which points to the vtable for that class.
During virtual function calls, the vptr is used to look up the function address in the vtable, allowing dynamic dispatch.

During virtual function calls, the vptr is used to look up the function address in the vtable, allowing dynamic dispatch.
