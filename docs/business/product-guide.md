# Product Skills Guide

8 skills covering the product development lifecycle: experimentation, analytics, user research, growth engineering, pricing, go-to-market, onboarding, and feature prioritization.

## Skill Map

| Skill | Directory | Focus |
|-------|-----------|-------|
| A/B Testing | `skills/product/ab-testing/` | Experiment design, variants, statistics, analysis |
| Analytics | `skills/product/analytics/` | Event tracking, funnel, cohort, retention |
| Feature Prioritization | `skills/product/feature-prioritization/` | RICE, ICE, MoSCoW, value vs effort |
| Go-to-Market | `skills/product/go-to-market/` | Launch strategy, positioning, channels, timing |
| Growth Engineering | `skills/product/growth-engineering/` | Acquisition, activation, retention, virality |
| Onboarding Flow | `skills/product/onboarding-flow/` | User activation, funnel optimization, drop-off |
| Pricing Strategy | `skills/product/pricing-strategy/` | Model, tiers, packaging, willingness-to-pay |
| User Research | `skills/product/user-research/` | Interviews, surveys, usability testing, JTBD |

## Decision Framework

```
Need to discover what to build?
  ├─ User Research — interviews, surveys, JTBD
  └─ Analytics — quantitative data, funnel analysis

Need to decide what to build next?
  ├─ Feature Prioritization — RICE, MoSCoW
  └─ A/B Testing — validate with data

Need to launch a product?
  ├─ Go-to-Market — positioning, channels
  ├─ Pricing Strategy — tiers, packaging
  └─ Onboarding Flow — activation

Need to grow?
  ├─ Growth Engineering — AARRR metrics
  └─ Analytics — track everything
```

## Product Lifecycle

```
┌─────────────────────────────────────────────────────────┐
│              DISCOVERY                                   │
│  User Research → Analytics → Feature Prioritization      │
├─────────────────────────────────────────────────────────┤
│              VALIDATION                                  │
│  A/B Testing → Pricing Strategy → User Research (again)  │
├─────────────────────────────────────────────────────────┤
│              LAUNCH                                      │
│  Go-to-Market → Onboarding Flow → Analytics              │
├─────────────────────────────────────────────────────────┤
│              GROWTH                                      │
│  Growth Engineering → A/B Testing → Analytics            │
└─────────────────────────────────────────────────────────┘
```
