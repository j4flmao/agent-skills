# Management Skills Guide

14 skills covering engineering management, project management, quality assurance, security operations, stakeholder communication, and team operations.

## Skill Map

| Skill | Directory | Focus |
|-------|-----------|-------|
| Alerting | `skills/management/alerting/` | On-call, escalation, policies, notification |
| BA (Business Analysis) | `skills/management/ba/` | Requirements, specifications, user stories |
| Cost-Benefit | `skills/management/cost-benefit/` | ROI, TCO, build vs buy, cost estimation |
| Hiring | `skills/management/hiring/` | JD writing, interview process, evaluation |
| OKR & KPI | `skills/management/okr-kpi/` | Goal setting, tracking, alignment, reviews |
| Pentesting | `skills/management/pentesting/` | Scope, methodology, reports, remediation |
| PM (Project Management) | `skills/management/pm/` | Planning, estimation, risk, tracking |
| QA | `skills/management/qa/` | Test strategy, test plans, manual testing |
| QC | `skills/management/qc/` | Quality control, inspections, checklists |
| Risk Management | `skills/management/risk-management/` | Identification, assessment, mitigation |
| Security | `skills/management/security/` | Policy, training, audits, incident management |
| Sprint Retro | `skills/management/sprint-retro/` | Retrospectives, action items, improvement |
| Stakeholder | `skills/management/stakeholder/` | Communication, reporting, expectation management |
| Team Rules | `skills/management/team-rules/` | Working agreements, code of conduct, norms |

## Decision Framework

```
Need to plan work?
  ├─ PM — project plan, timeline, milestones
  ├─ BA — requirements, specifications
  └─ Cost-Benefit — business case

Need to execute work?
  ├─ Team Rules — working agreements
  ├─ OKR/KPI — goals and metrics
  └─ Sprint Retro — continuous improvement

Need to ensure quality?
  ├─ QA — test strategy, test plans
  ├─ QC — inspections, checklists
  └─ Pentesting — security testing

Need to manage people?
  ├─ Hiring — recruitment process
  ├─ Stakeholder — communication plan
  └─ Risk Management — issue mitigation

Need to run operations?
  ├─ Alerting — on-call, incident response
  └─ Security — policy, training, audits
```

## Management Lifecycle

```
┌─────────────────────────────────────────────────────┐
│                  PLANNING                             │
│  BA (requirements) → Cost-Benefit → PM (plan)        │
│  → OKR/KPI (goals)                                   │
├─────────────────────────────────────────────────────┤
│                  EXECUTION                            │
│  Team Rules → Sprint Retro → QA/QC → Stakeholder     │
├─────────────────────────────────────────────────────┤
│                  OPERATIONS                           │
│  Alerting → Risk Management → Security               │
├─────────────────────────────────────────────────────┤
│                  GROWTH                               │
│  Hiring → OKR/KPI review → Cost-Benefit (retro)      │
└─────────────────────────────────────────────────────┘
```
