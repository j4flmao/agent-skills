# Conflict Resolution in Replication
## Algorithms and Formulations
Vector Clocks:
A vector clock of a system of N processes is an array/vector of N logical clocks.
$$ VC = [c_1, c_2, ..., c_N] $$

Rules:
1. Initially all clocks are zero.
2. Each time a process experiences an internal event, it increments its own logical clock in the vector by one.
3. Each time a process sends a message, it includes its vector clock.

Last-Writer-Wins (LWW):
Resolves conflicts by picking the change with the largest timestamp.
