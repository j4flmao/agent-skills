# Apache Iceberg Table Formats
## Algorithms and Formulations
Iceberg tracks individual data files in a table instead of directories. This allows for scalable metadata management.

## Data Schemas
```json
{
  "format-version": 2,
  "table-uuid": "fb072c92-a02b-11e9-ae9c-1bb7bc9cca94",
  "location": "s3://bucket/warehouse/table",
  "last-updated-ms": 1515100955770
}
```

## Code Examples
```python
import pandas as pd
# Iceberg read example
df = spark.read.format("iceberg").load("db.table")
```
