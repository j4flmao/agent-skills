# Batch Processing Patterns
## Best Practices
1. Use combiners to reduce network transfer between mappers and reducers.
2. Partition data to avoid data skew.
3. Compress map outputs using Snappy or LZO.

## Anti-patterns
- Small files problem (HDFS limitation)
- Memory-heavy reducers without spilling
